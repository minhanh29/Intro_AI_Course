This program find the connections between 2 stars based on the csv files
(2 stars are connected if they both look at the same movie or have a connection with other connected stars)

Open the program by typing this command in terminal or cmd
python3 degrees.py small

Enter two names in the people.csv file (in the chosen directory) to see their connections.

Example output:
Loading data...
Data loaded.
Name: Kevin Bacon
Name: Tom Cruise
1 degrees of separation.
1: Kevin Bacon and Tom Cruise starred in A Few Good Men
