This program predicts whether an online customer purchases a product.

1. Install scikit-learn
pip install -U scikit-learn

Check the module infor:
python3 -m pip show scikit-learn

2. Open the program by typing this command: 
python3 shopping.py [filename].csv 


Example output:
python3 shopping.py shopping.csv 
Correct: 4093
Incorrect: 839
True Positive Rate: 42.01%
True Negative Rate: 90.28%
